<!-- GETTING STARTED -->
## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com//Muhammad-Elgendi/Ersodli.git
   ```
2. Go to Ersodli directory
   ```sh
   cd Ersodli/
   ```
3. open index.html in browser
